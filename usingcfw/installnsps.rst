Install NSPs over USB
=====================

.. toctree::
   :maxdepth: 2
   :caption: Contents:
   :hidden:
   
   /usingcfw/installnsps/installnsps-goldtree
   /usingcfw/installnsps/installnsps-nsusbloader

Installing NSPs over a USB connection to your computer can be faster than first transferring to your SD card and then installing, plus it solves issues with FAT32 file size caps. This guide will teach you how to install NSPs over USB using Goldleaf.

.. note:: 
   If something goes wrong or you need help, check `troubleshooting </troubleshooting.html>`_.
